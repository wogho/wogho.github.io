# Contributing

Thanks for your interest in contributing! Here's how to get started.

## Development Setup

```bash
# Fork and clone the repo
git clone https://github.com/YOUR_USERNAME/awesome-portfolio-template.git
cd awesome-portfolio-template

# Install dependencies
npm install

# Start dev server
npm run dev
```

## Guidelines

- **Code style**: Follow the existing patterns. Use TypeScript strict mode.
- **Commits**: Write clear, concise commit messages.
- **PRs**: One feature/fix per PR. Include a description of what changed and why.
- **Tests**: Run `npm run build` before submitting to ensure no build errors.

## What to Contribute

- Bug fixes
- Accessibility improvements
- Performance optimizations
- New section components (with sample data)
- Translation improvements
- Documentation

## What NOT to Contribute

- Breaking changes to the config structure without discussion
- New npm dependencies without justification
- Personal portfolio content (keep sample data generic)

## Questions?

Open an issue with the `question` label.
